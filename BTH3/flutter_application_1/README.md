# Pick More Shop - TH3 Network Fetch Demo

This minimal Flutter app (Pick More Shop) demonstrates fetching product data from a public API and handling Loading / Success / Error states with a Retry button. The app displays household essentials and supports a simulated network error for testing the Error+Retry UI.

To run:

```bash
cd d:\ANDROI\BTH3\flutter_application_1
flutter pub get
flutter run -d edge --web-port=8083
```

Edit the student name/id in `lib/main.dart` to match the required AppBar format.
